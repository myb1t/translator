Online Translator

Simple and fast online translation tool for macOS.

Features

  • Automatic Clipboard Translation – just copy text
  • Text translation in any application that supports text selection
  • PopClip extension
  • Save translation to clipboard
  • Keyboard shortcuts

How to enable MyDictionary

Open System Preferences → Keyboard → Keyboard Shortcuts → Services
and enable the “MyDictionary” option.

Notes

Internet connection is required.
